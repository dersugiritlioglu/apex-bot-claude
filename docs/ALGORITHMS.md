# Algorithm Reference

## #1 Trend Rider — BULLISH
MACD crosses above signal → entry. EMA9 > EMA21 = full size. EMA9 < EMA21 = 50% size.
SL 5% · TP 10%

## #2 Momentum Surge — BULLISH + HIGH VOLUME
MACD cross + volume ≥ 1.5× 20-period avg required. No volume = skip.
SL 5% · TP 12%

## #3 Bear Bouncer — BEARISH
Price at lower BB (2σ) + RSI < 35. RSI < 30 → 75% size, else 50%.
SL 3% · TP 7%

## #4 Range Hunter — SIDEWAYS
RSI < 32 + BB %B < 0.20 = buy. RSI > 68 + BB %B > 0.80 = sell. Size 40%.
SL 2.5% · TP 5%

## #5 Volatility Shield — VOLATILE
EMA9/21 crossover only. Size 25%.
SL 4% · TP 8%
